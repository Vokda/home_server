Name Generator

This is a rewrite of my app Danielian Name Generator in perl. The app's old source code is archive away somewhere on my system and rather than finding it to fix its minor bugs I rewrote it. Besides, perl is nicer than java.

Current limitations:

  • No dynamic name lengths
  • Cannot change letter frequency
  • The same two letters will not occur in a row
Fixes:
  • More than three of the same letter will not occur anymore.
  • Removed unnecessary "work-around functions" for names that would be unpronounceable, e.g. "bkkkktv"
  • Previous point did not work anyway, name generated is pronouncable from the beginning now.
